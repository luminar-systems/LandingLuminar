# Luminar Systems — Landing

Sitio corporativo de **Luminar Systems**, casa de software guatemalteca.
Es una página **autocontenida** (`index.html`): fuentes de sistema, SVG y CSS/JS en línea, sin dependencias externas ni build.

## Desplegar

### Cloudflare Pages (recomendado)
1. Cloudflare → **Workers & Pages** → **Create** → **Pages** → **Connect to Git** → este repositorio.
2. Framework preset: **None** · Build command: *(vacío)* · Output directory: `/`
3. **Deploy**. Luego, en **Custom domains**, agrega `www.luminarsystems.com.gt`.

### GitHub Pages (alternativa)
**Settings → Pages → Build and deployment → Deploy from a branch →** `main` / `/ (root)`.
